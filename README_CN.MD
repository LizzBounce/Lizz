## 什么是Lizz
Lizz是一个基于Mixin注入的黑客端，使用Minecraft Forge 1.8.9，基于CCBluex的水影。

## 如何构建Lizz
1.下载[Lizz](https://github.com/LizzBounce/Lizz/archive/refs/heads/main.zip)源码

2.用IDEA打开此项目

3.在终端输入./gradlew build

4.在build/libs就可以找到本体了
